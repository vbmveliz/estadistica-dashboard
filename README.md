# 📊 Sistema de Estadísticas Dashboard

Este proyecto es un **dashboard de estadísticas desarrollado en PHP y MySQL**, utilizando **AdminLTE** para la interfaz gráfica.

El sistema permite gestionar información de estados, pendientes y estadísticas mediante un panel visual.

---

# 🚀 Características

- Sistema de autenticación (Login)
- Dashboard administrativo
- Gestión de estados
- Gestión de pendientes
- Visualización de estadísticas
- Preparación de materiales
- Interfaz basada en AdminLTE

---

# 🛠 Tecnologías utilizadas

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- AdminLTE
- Bootstrap
